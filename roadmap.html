    Array

    An Array is a linear data structure that collects elements of the same data type and stores them in contiguous and adjacent memory locations. Arrays work on an index system starting from 0 to (n-1), where n is the size of the array.

    String

    String is generally considered as a data type and is often implemented as an array data structure of bytes (or words) that stores a sequence of elements, typically characters, using some character encoding.

    Hash Table

    In a Hash Table, data is stored in an array format, where each data value has its own unique index value. Access of data becomes very fast if we know the index of the desired data.

    2 / 23

    Two Pointer

    Two Pointer is an approach in which we search a particular subarray that is satisfy the problem. It's much effective that a Brute / Naive approach.

    2 / 23

    Sliding Window

    Sliding Window is a technique which aims to reduce the use of nested loop and replace it with a single loop, thereby reducing the time complexity.

    2 / 23

    Binary Search

    This is an effective search method that find the index of target number in an array. It have advantage over linear search method as it's TC - O(log n) but array must be in sorted fashion.

    2 / 23

    Stack

    Stack is an linear data structure which follows a particular order in which the operations are performed. The order is LIFO (Last In First Out).

    2 / 23

    Queue

    Queue is an linear data structure that is open at both ends and the operations are performed in First In First Out FIFO (First In First Out) order.

    2 / 23

    Depth First Search

    Depth First Search is a recursive traversal algorithm for searching all the vertices of a matrix , tree or graph data structure. It starts from the first node of graph G and then goes to further vertices until the goal vertex is reached.

    2 / 23

    Linked List

    Linked List is a linear data structure, in which the elements are not stored at contiguous memory locations. The elements in a linked list are linked using pointers.

    2 / 23

    Breadth First Search

    Breadth First Search is an algorithm in which we traverse the matrix, tree or graph data structure by using queue, it remember the next point to start a search, when a dead end occurs in any iteration.

    2 / 23

    Tree

    Tree is a hierarchical data structure defined as a collection of nodes. Nodes represent value and nodes are connected by edges. It have a root and have nodes, every node have a parent node except root node.

    2 / 23

    Tries

    Tries are an extremely special and useful hierarchical data structure that are based on the prefix of a string. It work similar as tree.

    2 / 23

    Heap / Priority Queue

    A Heap is a special Tree-based data structure in which the tree is a complete binary tree where the root-node key is compared with its children and arranged accordingly.

    2 / 23

    Recursion

    Recursion is way in which a function calls itself indirectly or directly, It's generally used when the answer to a larger problem could be divide in terms of smaller problems.

    2 / 23

    Back Tracking

    Backtracking is an algorithmic technique for solving problems recursively by trying to build a solution incrementally, one piece at a time, removing those solutions that fail to satisfy the constraints of the problem at any point of time.

    2 / 23

    Graph

    A Graph is a non-linear data structure consisting of vertices and edges. Each vertex could be connect to multiple vertex with the help of edge.
